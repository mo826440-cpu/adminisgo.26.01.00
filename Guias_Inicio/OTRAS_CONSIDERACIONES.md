Este archivo lo usaremos para ir verificando consideraciones que vayan surgiendo durante la creación del proyecto.

Quiero que cada vez que se cree o actualice el proyecto, para cada programación se tengan en cuenta los archivos dentro de la carpeta Guia_Inicio.

Tablas visibles de los registros dentro de la app. Al registrar datos dentro de la app, estos datos luedo se van a mostrar en una tabla visible dentro de cada sesión correspondiente.
 - Quiero que los títulos de cada columna siempre estén visibles, ejemplo, al 
   scrolear hacia abajo, quiero que se sigan viendo. 
 - Quiero que los registros visibles para cada tabla no sean más de 100, pero con la posibilidad de ir viendo por págins, ejemplo, página 1 muestra los últimos 100 registros cargados, página 2 los otros 100 y asi sucesivamente.

Navegación entre las diferentes sesiones o diferentes páginas:

  - Quiero que se pueda navegar mediante opciones desde una barra lateral, ejemplo: 

    * Dashboard
    * Referencias: - Categorias
                   - Marcas
                   - Clientes
                   - Proveedores
                   - Productos
    * Usuarios
    * Compraas
    * Ventas
    * Reportes
    * Configuraciones
    * Mantenimiento
